# Backup and Recovery

## System Backup
 - **Creates a system backup.**
   ```shell
        mksysb
   ```
 - **Saves volume group data.**
   ```shell
        savevg
   ```
  - **Archives files.**
   ```shell
        tar
   ```

  ## Restore
  - **Restores a volume group.**
   ```shell
        restvg
   ```

  - **Restores files from backups.**
   ```shell
         restore
   ```
